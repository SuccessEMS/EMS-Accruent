Exchange Integration

EMS Integration to Exchange integrates EMS Everyday User applications, such as EMS Mobile App and EMS Web App, with Microsoft® Exchange. This component enables everyday users to view the availability of both meeting rooms and attendees, and send Outlook® meeting invitations, all from within EMS Mobile App and EMS Web App. This integration requires that you enable Exchange Web Services Impersonation, and that you have licensing for EMS, EMS Web App, and EMS Integration to Microsoft® Exchange. EMS works with Exchange Web Services (EWS) through the EMS Web service to retrieve information on attendee free / busy status. By connecting EMS to EWS, you can:

  • Invite attendees from both Outlook and the EMS Web App.
  • Keep your Outlook calendar in sync with any EMS reservations.
  • Support the Exchange Integration Web service.
