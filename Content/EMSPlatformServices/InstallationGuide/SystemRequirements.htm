System Prerequisites and Requirements for EMS Platform Services

The following information is necessary to successfully install EMS Platform Services. See Also: Licensing Requirements.

As of Update 28, users are required to deploy EMS Platform Services with ASP.NET Core, rather than the HTTP Platform Handler.

EMS Platform Services Prerequisites and Requirements

Install the ASP.NET Core

As of Update 28, users are required to deploy EMS Platform Services with ASP.NET Core, rather than the HTTP Platform Handler.

  1. Download the .NET files.

  2. Choose .NET Core version 2.1.

  3. In the right-hand “Runtime” column, select ASP.NET Core/.NET Core: Runtime & Hosting Bundle.

    ASP.Net handler is a part of the Runtime & Hosting Bundle.
