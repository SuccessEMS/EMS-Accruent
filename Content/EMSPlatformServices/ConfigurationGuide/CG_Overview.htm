EMS Platform Services Configuration Guide

From the EMS Platform Services Admin Portal, you can clear the cache, view license and application information, configure authentication methods, manage integration clients, and view logs. You can configure settings for EMS Platform Services from the following tabs: 

  • Home—Perform Admin Functions

  • Integrations—Manage EMS Integrations

  • Roles—Create New Roles or Edit Existing Ones

  • Logs—View Global and Integration Logs

  • Header

  • OpenID

  • SAML

  • Auth Keys—Create New Keys

  • Calendaring—Enable GSuite Integration

  • Conferencing—Skype for Business
