System Prerequisites and Requirements for ERI

Prerequisites

You must meet the following prerequisites to use Exchange Room Integration (ERI):

  • A valid license for ERI and a valid sync limit (1 or more)

    To determine if you have a valid ERI sync limit, verify that the value for the Exchange Room Sync Limit is 1 or more in DEA Customer Relationship Manager. The Exchange Room Sync Limit is the number of rooms that can be configured with Exchange mailboxes.

  • EMS Platform Services minimum version required: Update 33

  • Meet system requirements

    •  sync requirements

      • Exchange version supported: Microsoft Office 365

        Hosted, Hybrid, and On-prem are note supported because of Microsoft limitations. For more information, refer to the Microsoft's documentation.

    • sync requirements

      • Microsoft Exchange 2016; 2013; 2010, SP3

      • Microsoft Office 365

  • Admin user within your Office 365 account

  • Access to EMS Platform Services Admin Portal

System Requirements

The latest version of EMS software (V44.1) must be installed before you begin Installing Exchange Room Integration.
