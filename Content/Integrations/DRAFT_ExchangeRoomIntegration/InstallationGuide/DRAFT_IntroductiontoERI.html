Introduction to Exchange Room Integration (ERI)

What Is ERI?

Exchange Room Integration (ERI) is a simplified integration tool that allows enhanced integration of resources and technology during the EMS booking process for managed spaces. Exchange Room Integration simplifies the booking process and provides a cohesive and powerful way to book and manage resources. 

Features:

  • Allows integration of third-party technology (e.g., video conference [VC] device or display panels) to a booking in EMS by linking to a Microsoft® Exchange resource mailbox

  • Automatically includes fixed-room resources in the room booking (e.g., collaboration devices) through the Microsoft® Exchange resource mailboxes

  • Allows integration of non-fixed resources (e.g., Mobile Microsoft Surface Hubs) during the booking process

  • Provides bi-directional syncing— and

  • Allows syncing of bookings with rooms linked to Exchange mailboxes from the following EMS products: 

    • EMS Desktop Client (new as of Update 33)

    • EMS Room Sign App (new as of Update 33)

    • EMS Kiosk App (new as of Update 33)

    • EMS Web App

    • EMS for Outlook Add-in

    • EMS Mobile App

    For information on the types of booking data that can be synced between EMS and Exchange using ERI, see ERI Interoperability Between EMS Products and Exchange.
