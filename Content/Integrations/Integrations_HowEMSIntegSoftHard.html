How Does EMS Integrate with Other Software and Hardware?

We provide optional integration tools, services, and APIs to enable you to connect with your existing systems and with software and hardware you already have. EMS provides a wide range of tools and services to enable fast integration and data sync during the implementation process. 

Integration Diagram

Web Browsers and Mobile Devices

EMS Web App enables web users to view and book space on-the-go using any web browser application. EMS Mobile App provides similar functionality for smaller mobile devices such as smartphones and tablets. As an administrator, you define web users' security access and booking options in EMS Desktop Client, including guests and temporary visitors.

Toolkits and Data Interfaces

EMS software is infinitely configurable; to simplify this effort, we provide toolkits and web services that enable you to import and export real-time data between systems and initiate automated bookings with other software applications. Using the EMS tools and services, you can:

  • Eliminate duplicate information and planning mistakes.
  • Provide timely updates of any new or altered user information.
  • Link and sync calendars with scheduling.
  • Coordinate service and resource usage with the proper systems.
  • Ensure the right users have the right privileges.

Reporting

EMS provides more than 110 reports and a custom query builder for use throughout the meeting and event lifecycle—from setup and invoicing to billing and utilization analysis. These "one-way integration" reports generate static reservation, service, billing, resource, and other information in .xml, .xls, comma-delimited, and .txt formats. This data can be read and parsed by many other software applications. These reports are categorized as follows:

  • Daily: Summary, service, and other reports that describe activities over one or more days.
  • Sales: Details of sales associated with reservations, resources, and more.
  • Statistics: Insights you need to understand use and organizational activity.
  • Exception: Clarifies the incorrectly run or allocated resources, templates, or systems.
  • Billing: All reports for transactions, invoices, and other statements.
  • Hoteling: Reports specific to office hoteling practices and QR codes for room cards.
  • Other: Summary reports on cancellations and reservations.

You can send reports according to whatever frequency, time, time zone, and reporting period you choose, and you can customize the sender, subject, and message information. To set up Automated Reports, your organization will need to install and configure the EMS Automated Report service.

QR Codes

You can associate rooms with QR Codes: EMS Desktop Client provides a report that generates the codes and associates them with the designated rooms. You can print room cards that attendees can then scan or swipe (on room signs or using mobile devices) to see check-in to meetings.

APIs

You can use our APIs to develop your own custom solutions for taking data from EMS and using it elsewhere, to display information on calendars, websites, digital signage, and to integrate with third-party service systems.

Microsoft® Exchange Integration

EMS Integration to Microsoft® Exchange integrates EMS Everyday User applications, such as EMS Mobile App and EMS Web App, with Microsoft® Exchange. This component enables everyday users to view the availability of both meeting rooms and attendees, and send Outlook® meeting invitations, all from within EMS Mobile App and EMS Web App. 
See Also: Introduction to Integration to Microsoft® Exchange

Security: Integrated Windows Authentication

EMS integration with Active Directory (AD / LDAP) lets your organization use your HR data, such as existing user records and accounts, as the sign-in credentials for EMS. As employee information and status change, this integration updates to EMS at a chosen frequency, reading groups and assigning roles based on that information. Additionally, you can easily reference your existing AD / LDAP information as Single Sign-On (SSO) authentication. 

See Also: Integrated Windows Authentication

Digital Signage: Video Displays, Tablets, and Room Signs

Placed outside the entrance to meeting rooms, wall-mounted digital signs with EMS software allow employees to book themselves into reservable spaces. EMS provides the following, optional licenses to integrate with digital tablets and room signs. See Also: Integrations in EMS Platform Services.

EMS Kiosk

Run on a touch-screen display unit, EMS Kiosk is a powerful yet easy-to-use digital room sign and interactive tool for reviewing room availability, booking space, searching for a meeting location and more. Mount the unit outside a room and/or make an EMS Kiosk available in a lobby or entryway. Pair with the optional EMS Floor Plan module and give users an added measure of convenience as they see reserved/available space on a facility diagram, making it even easier to find  and hold meeting space. 

See Also: EMS Kiosk App

EMS Room Sign App

This web-based application turns your Apple iPad or Android tablet into a digital room sign, displaying real-time meeting information and at-a-glance room availability. It supports Apple, Android, and Microsoft tablets, and integrates with popular room sign products, such as Crestron Connected Scheduling Application and Steelcase RoomWizard, a separate EMS module.

See Also: EMS Room Sign App

RoomWizard

The EMS-RoomWizard Interface is an optional module for communicating room reservation information between EMS and Steelcase RoomWizard. A RoomWizard is a wall-mounted unit that is associated with a particular room and used to schedule and view events in it.

See Also: Integration to RoomWizard

Video Displays

The EMS Video Display Interface (VDI) is an add-on component that exports meeting and event information to a video display system such as Janus, Target Vision, and so on. 

HVAC

Unused space often has heating, cooling, and other utilities continually running — equating to a large, unnecessary expense. Compliant with green initiatives and LEED certification, EMS API and Facilities Management Toolkit help organizations reduce costs: they can be configured to integrate with third-party Heating, Ventilation and Air Conditioning (HVAC) systems (such as EVENTS2HVAC™ by Streamside Solutions) to automate heating and cooling based on room usage.

Geolocation, Mapping, and Floorplans 

integrates with mobile devices' geographic mapping functionality to enable users to locate nearby rooms and to book space and check into meetings. For more detailed EMS applications like EMS Web App and EMS Glance, building floorplans and room layout images can be configured to visually show user location, room proximity and location on a campus and on a floor, and room drawings.
