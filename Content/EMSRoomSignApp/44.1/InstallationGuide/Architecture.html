Architecture for EMS Room Sign App

The application is comprised of four main components:

  1. EMS Room Sign App Device: Device with a supported browser installed that will access the web application. A particular device is tied to one specific room in EMS.
  2. EMS Room Sign App Web Application: EMS Room Sign web application.
  3. EMS Room Sign App Web Service API: Web service API that manages all communication between the web application and your EMS database. This component is only required for Update 29 or earlier.

    For Update 30Update 30 or later, the EMS Room Sign App web service API is no longer necessary because the API for EMS Room Sign App has been moved into the application to improve performance.

  4. EMS Room Sign App Settings DLL: Component installed with your EMS Client application (.exe) that allows your EMS Administrators to configure the EMS Room Sign App.
