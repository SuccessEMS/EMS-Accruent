Optional Integrations and Features for EMS Software

EMS Software provides optional integration tools, services, and APIs to enable you to connect with your existing systems and with software and hardware you already have. EMS provides a wide range of tools and services to enable fast integration and data syncing during the implementation process.

Integrations Available for EMS Software

  • Exchange Room Integration (ERI)

  • Human Resources Information Systems (HRIS)

  • Microsoft Exchange Integration

  • Video Conferencing

  • Skype for Business

Optional Features Available for EMS Software

  • EMS API

  • EMS Floor Plans

    • EMS Floor Plans Installation Guide

    • EMS Floor Plans Configuration Guide

  • EMS Facility Management Toolkit

  • EMS Human Resources Toolkit

  • Active Directory

  • Cisco TMS Interface

  • EMS Integration to RoomWizard

  • Polycom Interface

  • Services and Components
