Security Considerations

Transport Security

The RoomWizard units can be configured to communicate with the EMS-ROOMWIZARD Interface using SSL.

Private Events

EMS Events that are considered not ‘Display on Web’ are treated as private events with respect to the RoomWizard unit. If an EMS Event meets any of the below criteria the event will be considered private and will display on the unit with a Purpose (Event Name) of ‘Private’.

Criteria for Private Events:

  • Booked Room is not ‘Display On Web’
  • Booking Status is not ‘Display On Web’
  • Booking Event Type is not ‘Display on Web’
