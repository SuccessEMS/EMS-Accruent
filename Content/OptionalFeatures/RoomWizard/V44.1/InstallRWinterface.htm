Install the EMS – RoomWizard Interface

  1. Obtain the EMS_Install.sql AND the EMSMaster_Install.sql scripts.
  2. Open Microsoft SQL Server Management Studio.
  3. Within Microsoft SQL Server Management Studio, select File > Open > File… and locate EMSMaster_Install.sql.
  4. Select your EMS_Master database and execute the query (Query > Execute). When query execution has completed, a ‘Command(s) completed successfully’ message will display in the Results section.
  5. Within Microsoft SQL Server Management Studio, select File > Open > File… and locate EMS_Install.sql. 
  6. Select your EMS database and execute the query (Query > Execute). When query execution has completed, a ‘Command(s) completed successfully’ message will display in the Results section.
