Install the Polycom Activity DLL on Users' Machines

  1. On each client machine that needs to access the Polycom Activity area, paste a copy of the EMS.PolycomActivity.dll file into the folder that contains your EMS application files (typically C:\Program Files\EMS).

  2. Once the EMS.PolycomActivity.dll is placed in the correct folder on the user's machine, the Polycom Activity button will show on the toolbar in the EMS application (assuming that you have updated your EMS Registration with the appropriate licensing information).
