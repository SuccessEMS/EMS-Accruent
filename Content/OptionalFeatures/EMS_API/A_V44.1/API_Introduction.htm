Introduction to EMS APIs

This guide is intended for programmers that need detailed information about our web service based API. Use of this guide assumes you are familiar with the following:           

  1. XML
  2. Web services
  3. Query or SOAP
  4. One or more programming languages

If an offsite external third party integrator (i.e. outside of your network) requires access to the EMS API, the EMS API Web Service will need to be installed as a public facing website.
