Copy Booking Details Wizard

You use the Copy Booking Details Wizard to copy details from one booking to another booking under the same reservation.

  1. Open the parent reservation in the Navigator. 

  2. In left pane of the Navigator, select the reservation or booking with which you want to work. 

  3. In the Bookings pane, click Tools, and then click Copy Booking Details Wizard. 

    The Copy Booking Details Wizard appears, listing all the bookings for the selected reservation.

    You can change the reservation from which to copy details by clicking the Search icon. You can narrow the list by selecting a Category. You can deselect the Copy Inactive Resources/Rooms if you want to carry over these items.

  4. Select the booking(s) for which you want to copy details and click Next. 

    The next window lists the details you can copy.

    Use the Hide Canceled Bookings and Hide Old Bookings check boxes to manipulate the list.

  5. Select individual or multiple booking details (click Select All to select all the displayed bookings in a single step), and then click Finish. 

    A message confirms the copy.

  6. Click OK to close the message and return to the Navigator.
