Configuration > Other

This section guides you in configuring other data types, under Configuration > Other.

From the Configuration > Other menu, you can access and configure the following data types:

  • Event Types
  • Group Types
  • Reminder Types
  • Comment Types
  • Messages
  • Special Dates or Holidays
  • User Defined Fields (UDFs)
  • Reservation Sources
  • Cancel Reasons
  • Attachment Types
  • Word Merge Documents (Optional Module)
  • Calendar Styles
