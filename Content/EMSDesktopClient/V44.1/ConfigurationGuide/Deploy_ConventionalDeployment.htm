Conventional Deployment

What is Conventional Deployment? 

The conventional deployment method is delivered in a typical MSI file. This file can be run directly on a computer, or installed via an administrative push. Standard toolsets for working with MSI files can be used for deployment. If your organization already has polices and systems in place for pushing applications, this might be the preferred way to deploy the EMS Desktop Client. If you are looking for an easier way to deploy the EMS Desktop Client, see Understand EMS Desktop Client Web Deployment .
