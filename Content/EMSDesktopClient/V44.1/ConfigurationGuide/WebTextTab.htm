Help Text Tab

To configure help text for an everyday user process template, follow these steps:

  1. Open the Help Text tab of the Everyday User Process Template dialog box.

Help Text Tab

  1. By default, web text items inherit their settings from the global Web Text list. (See Also: Configure Web Text.) You can use the list of help text items on the Help Text tab to override the settings for this particular template if needed.
    • To edit the value for a web text item, select the item, and then click Edit.
    • To reset the value for an item to its default global value, select the item, and then click Use Global.
  2. Continue with any other configuration for the templates as needed; otherwise, click OK to save the changes to the template. For an Everyday User to be able to view and use this template, you must assign the user to the template. See Assign Templates to Everyday Users.
