User-Defined Fields (UDF) Tab

To configure user-defined fields for an everyday user process template, follow these steps:

  1. Open the User Defined Fields tab of the Everyday User Process Template dialog box.

User-Defined Fields Tab

  1. On the Available list, select the question or CTRL-click to select the multiple questions that users who are attached to this Everyday User template must answer when reserving space, and then click the Move button (>) to move the selected questions to the appropriate Selected list.
    • Move questions that require an answer into the Required pane.
    • Move questions that are optional into the Optional pane.

If the appropriate UDF is not available, you can configure it. See Configure User Defined Fields (UDFs).

  1. Continue with any other configuration for the templates as needed; otherwise, click OK to save the changes to the template. For an Everyday User to be able to view and use this template, you must assign the user to the template. See Assign Templates to Everyday Users.
