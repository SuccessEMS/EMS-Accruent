Configure Room Types

A room type is the class or category of the room (e.g., a meeting room, an auditorium, a workspace, a multi-purpose room, etc.). Users can use room types to filter search results.

  1. On the menu bar, click Configuration > Facilities > Room Types.
    The Room Types window opens. This window lists all the room types that are currently configured in your database.

    Room Types Window

  2. Click New. 

    The Room Type dialog box opens.

    Room Type Dialog

  3. In the Room Type field, enter a name or description for the new room type. 

    The room type can be a maximum of 50 characters, including spaces.

  4. If you want all Everyday Users to see this room type when they are searching for available space, leave Available to Everyday Users checkbox selected.

  5. If you have EMS for Google Calendar, you can choose whether to display an image for the room type with one of the following options:

    Currently, the image options on the Room Types tab only work with EMS for Google Calendar. If you choose to display images for room types, the images will only appear in EMS for Google Calendar. These images will not appear in EMS Web App, EMS Mobile App, and so on.

    • No Display Image: Select this option to have no image displayed for the room type.

    • Select from default display images: From the drop-down list, select a default image to display for the room type.

    • Use custom display image: Select this option to upload an image to display for the room type. The recommended aspect ratio is 1:1 (square). The maximum image size is 300 x 300, and the image file must be a .png, .jpg, or .svg.

  6. Click OK. 

    The Room Type dialog box closes, and you return to the Room Types window with the newly configured room type automatically selected.

An Override Room is used in EMS to loosely define useable space that is not formally configured. See Also: Configure Override Rooms.
