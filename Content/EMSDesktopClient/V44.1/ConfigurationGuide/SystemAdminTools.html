System Administration > Tools

You have several tools at your disposal in EMS Desktop Client to assist you in your administrative duties. These include a tool to update various group attributes, such as the pricing plans or account name. You can also manage saved reports, view logged-in users, and if necessary, log these users off your EMS system, email logged-in users, and delete old data in your EMS database.

You can access these system administration tools under System Administration > Tools.

The following topics provide information about the tools found in System Administration:

  • Change Group Wizard
  • Change Reservation Wizard
  • Room Administration Wizard
  • Manage Saved Reports
  • Override Existing Bookings Utility
  • Import Reservation Utility
  • View and Email Logged-In Users
  • Delete Old Data
  • Currency Configuration Wizard
  • Booking Move/Change Status Wizard
  • Video Display Interface
  • Migrate Data
