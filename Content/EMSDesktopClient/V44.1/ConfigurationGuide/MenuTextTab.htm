Menu Text Translations Tab

You need to provide information on the Menu Text Translations tab only if a foreign language translation has been defined for Everyday User templates.

To configure menu text translations for an everyday user process template, follow these steps:

  1. Open the Menu Text Translations tab of the Everyday User Process Template dialog box.
  2. For each translation that is displayed on the tab, click in the Text field, and then enter the appropriate translation for the menu text (the menu text or link that an EMS Web App user sees when requesting a room using this Everyday User process template).
  3. Continue with any other configuration for the templates as needed; otherwise, click OK to save the changes to the template. For an Everyday User to be able to view and use this template, you must assign the user to the template. See Assign Templates to Everyday Users.
