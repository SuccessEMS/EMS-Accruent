Define Production Areas and Production Items for a Catering Resource

You use production items to define the quantity of components that make up a resource. For example, a resource called Box Lunch contains the following productions items—sandwich (quantity =1), chips (quantity =1), cookie (quantity = 2), and bottled water (quantity =1). If an order for 20 boxed lunches is placed, then the kitchen must produce 20 sandwiches, 20 bags of chips, 40 cookies, and 20 bottled waters. To identify the areas of the kitchen that produce each item, you can define production areas. The Delicatessen area might produce the sandwich, chips, and bottled water, and the Bakery area might produce the cookies.

If a category has been defined as using production items, then at the time that you are configuring a resource for the category, you can define the resource as a production item, or you can manually define resources as production items for the category at a later date.

This topic provides information on the following:

  • Define a New Production Area
  • Define a New Production Item
  • View Production Items and Production Areas

Define a New Production Area

  1. On the EMS menu bar, click Configuration > Resources. The Resources drop-down list contains all of the currently configured categories.
  2. From the Resources drop-down list, select a category (typically a catering type resource) for which to define a new production area. The Resources window opens, listing all active resources for the selected category.

    Resources Window

  3. Click the Prod. Items button on the Resources window. The Production Item window opens.

    The Prod. Items button will not be available if the resource category has not been defined to allow Production Items. Only Catering and Resources with or without Service Orders allow Production Items (see Configure Categories).

    Production Items Window

  4. From the Production Items window, click the Prod. Areas button. The Production Areas window opens and lists active Production Areas that have been configured for the selected category and that have a status of Active.

    Production Areas Window

    To see all production areas for the category that are in your EMS database regardless of status, under Show, click Inactive.

  5. Click New. The Production Area dialog box opens.

    Production Area Dialog Box

  6. Enter a description or name for the production area. The description can be a maximum of 50 characters, including spaces.
  7. Leave the Inactive check box blank to add the production area as an active area. Select the Inactive check box to inactivate the production area.
  8. Click Spelling if you want to spell check the description before you save the production area.
  9. Click OK. The Production Area dialog box closes. You return to the Production Areas window with the newly configured production area automatically selected.

Define a New Production Item

  1. On the EMS menu bar, click Configuration > Resources. The Resources drop-down list contains all of the currently configured categories.
  2. From the Resources drop-down list, select a category (typically a catering type resource) for which to define a new production area. The Resources window opens, listing all active resources for the selected category.
  3. Click the Prod. Items button on the Resources window. The button will not be available if the resource category has not been defined to allow Production Items. Only Catering and Resources with or without Service Orders allow Production Items (see Configure Categories). The Production Item window opens.

    Production Items Window

    You can change which production items are displayed by doing the following: 

    • To filter the displayed production items based on the Production Area, from the Production Area drop-down list, select a production area.
    • To view all production items that have been defined for the category regardless of status, under Show, click Inactive. 
    • To filter the list by item name, in the Filter field, enter a keyword. This search term is not case-sensitive, but your search is limited to the exact order of the characters in the string, and it must begin with the information for which you are searching. For example, a search string of "Cold" returns Cold Boxed Lunches, but not Sandwiches (Cold).

  4. Click New. The Production Item dialog box opens.

    Production Item Dialog Box

  5. In the Description field, enter description or name for the production item. The description can be a maximum of 50 characters, including spaces.
  6. If you want to select a production area for the item, select a production area from the Production Area drop-down list.
  7. Click OK. You return to the Production Items window with the newly configured production item automatically selected.
  8. Click Close on the Production Items window to close the window and return to the Resources window.
  9. On the Resources window, select the resource for which you are configuring the production items, and then click Edit. The Resource dialog box opens.
  10. Open the Production Items tab for the resource, and for each appropriate production item, enter the quantity.
  11. Click OK to close the Resource dialog box and save the resource with its production items.

View Configured Production Items and Production Areas

  1. On the EMS menu bar, click Configuration > Resources. The Resources drop-down list contains all of the currently configured categories.
  2. From the Resources drop-down list, click a category (typically a catering type resource) for the production item or area that you want to view. The Resources window opens, listing all active resources for the selected category.

    To see all resources that have been configured for a category, regardless of status, under Show, click Inactive.

  3. Click Prod. Items. The Production Item window opens, listing all active production items that have been defined for a category. From the Production Item window, you can view, define, edit, and delete production items.

    The Prod. Items button will not be available if the resource category has not been defined to allow Production Items. Only Catering and Resources with or without Service Orders allow Production Items (see Configure Categories).

    You can change which production items are displayed by doing the following: 

    • To filter the displayed production items based on the Production Area, from the Production Area drop-down list (Production Item window), select a production area.
    • To view all production items that have been defined for the category regardless of status, under Show, click Inactive (Production Item . 
    • To filter the list by item name, in the Filter field, enter a keyword. This search term is not case-sensitive, but your search is limited to the exact order of the characters in the string, and it must begin with the information for which you are searching. For example, a search string of "Cold" returns Cold Boxed Lunches, but not Sandwiches (Cold).

  4. Click Prod. Areas to open the Production Areas window. From this window, you can view, define, edit, and delete production areas.
