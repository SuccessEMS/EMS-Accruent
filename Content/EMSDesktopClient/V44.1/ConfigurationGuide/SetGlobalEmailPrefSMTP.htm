Set the Global Email Preference to SMTP and My Personal Email Preference to Outlook

Set the Global Email Preference to SMTP

To set the global email preference to SMTP, follow these steps:

  1. Log in to EMS Desktop Client as an Administrator.
  2. Select System Administration > Settings > Email Settings.
  3. Select Use SMTP.
  4. Enter your SMTP information.

Set Your Personal Email Preference to Outlook

To set your personal email preference to Outlook, follow these steps:

  1. Select Settings > Preferences.
  2. De-select Use Global Email Settings.
  3. Select Use Microsoft Outlook.
