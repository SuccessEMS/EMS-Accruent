Defaults Tab

To configure the default settings for an everyday user process template, follow these steps:

  1. Open the Defaults tab of the Everyday User Process Template dialog box.

Defaults Tab

  1. Enter the default values for Event Type, Event Name, Building, Event Time Start/End, and Time Zone. These defaults values automatically populate the correct fields on the Room Requests page, but a user can edit them as needed.

If the appropriate event type is not available, you can configure it (see Configure Event Types). If the appropriate building is not available, you can configure it (see Configure Buildings).

  1. Continue with any other configuration for the templates as needed; otherwise, click OK to save the changes to the template. For an Everyday User to be able to view and use this template, you must assign the user to the template. See Assign Templates to Everyday Users.
