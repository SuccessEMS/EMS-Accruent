API Call

 

GetUserDefinedFields SOAP API Call Added

The call will take “ParentID” and “ParentType” parameters. Results will be sorted in sequence, then parent/child order. Sequence, Field, Value, UDFID, ParentUDFID (this would be zero unless the UDF is a child UDF) responses are returned.

Fix

Area

Description

GetGroups

GetGroups API Call Not Returning Groups That Have Email Address Set To NULL

The GetGroups API was not returning groups that had blank email addresses, so for example importing a list of users may have been missing those with no email addresses. 

Fix: Modified functionality so that groups with blank email addresses are returned by the API call.

Area

Description

GetBookingHistory

Additional Support Required When Integrating Third-Party Room Signs

Some customers required additional support when integrating with third-party room signs that required combo room information.

Fix: Added the GetBookingHistory API Call, which provides the combo room information including Room ID to support  additional room-sign integration.
