Fixes

Area

Description

VC Bookings

Ending a VC Booking Ended All Bookings in that Reservation

Users ending a VC booking in the EMS Room Sign App was affecting all bookings tied to that reservation.

Fix: Modified functionality so that a non-host ending a VC booking only affects that booking.

Internal Reference # 36631
(For EMS Internal Use Only)
 

Reserve Button Disabled Close to Midnight

Half an hour or less before midnight, the reserve button was disabled and users could not make a reservation before midnight.

Fix: Modified functionality so users can use the reserve button to make a reservation if the room is available and the duration for the new booking allows the user to book the remaining time before midnight.

Internal Reference # 36857
(For EMS Internal Use Only)

Mimo Device

 

Users Could Not Select Profile and Room on Mimo

The settings modal was unresponsive using the EMS Room Sign App on a Mimo device and users were unable to apply a profile and select a room.

Fix: Modified functionality so that the settings modal is functioning properly.

Internal Reference # 37885
(For EMS Internal Use Only)
