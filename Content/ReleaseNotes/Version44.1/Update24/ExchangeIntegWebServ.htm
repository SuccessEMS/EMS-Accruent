Installation

Web.config File Improperly Defaulting User Value

During installation, the web.config file is defaulting the EMS user value to false instead of true. For most customers, this should be changed to ‘true’.

<databaseConnections>

<add name="deaConnection" useEmsUser="true" useDetailedLogging="true" defaultCommandTimeout="120"/>
