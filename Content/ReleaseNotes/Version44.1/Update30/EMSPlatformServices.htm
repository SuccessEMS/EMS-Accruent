Enhancements

Area

Description

Skype for Business

 

 

 

Display Only Region-Specific Dial-in Information

Skype for Business meeting invitation information generated through the EMS Web App now only displays the dial-in numbers for the applicable region with which the user belongs along with a link to reach the full dial-in number list.

EMS Internal Reference # 41481
(For EMS Internal Use Only)

"Try Skype Web App" URL and Text Added to Skype for Business Information

The link to "Try Skype Web App" has been added to the meeting invitation information generated when a meeting is initiated in the EMS Web App.

EMS Internal Reference # 41484
(For EMS Internal Use Only)

Enhanced Formatting

The font, size, and color of Skype for Business information that appears in the meeting invitation when a meeting is initiated from EMS Web App now matches the native Skype for Business add-in exactly.

EMS Internal Reference # 41486
(For EMS Internal Use Only)

Inclusion of Custom Footer Text If Configured within Skype for Business

If an organization has configured HTML-supported custom footer text within the Skype for Business application, the meeting invitation generated through the EMS Web App will now contain the custom footer text. The code indicating that the meeting invite is a Skype meeting is added to enable “Join Skype” on video conferencing devices.

EMS Internal Reference # 41488
(For EMS Internal Use Only)
