Fixes

Area

Description
 

Booking Time Not Updating When Booking Time Changed

When users dragged a booking to another time on the calendar, the booking time was not updating.

Fix: Modified functionality so that the booking time updates following a modification to the booking time.

Internal Reference # 38601
(For EMS Internal Use Only)
 

Service Times Not Updating When Booking Time Changed

When users modified a booking, the service order times were not updating.

Fix: Modified functionality so that the service times are updated following a booking time change.

Internal Reference # 38492
(For EMS Internal Use Only)
