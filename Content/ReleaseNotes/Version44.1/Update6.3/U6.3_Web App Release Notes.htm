Fixes 

Area

Description

New Outlook Booking

Edit/Delete Icons Showing Incorrectly for Multiple Rooms for Outlook Bookings

When users booked a new reservation using an Outlook template that requested multiple rooms for a single event, Edit/Delete icons showed next to two out of three of the rooms when only one should have been editable. This also occurred for multiples of 5 and 6 locations per occurrence.

Fix: Modified the display to ensure that only one room out of 3, 5, and 6 locations can be edited or deleted per single Outlook booking. 

24610

Rooms With Different Time Zones Display Incorrectly in Attendee Calendar for Outlook Bookings

When attendees viewed their calendars for Outlook events that had multiple locations in different time zones, the timeline and red time bars displayed incorrectly in the location time zone while the even displayed in the user's time zone, so the grid appeared to be misaligned.

Fix: Modified the display so that the timeline and red time bars display correctly on attendees' calendars so that they align with the event times in the grid.

24606

Edit Outlook  Booking

Timelines, Edit/Delete Icons, and Booked Locations Display Incorrectly on Calendar for Outlook Bookings

When users booked a new single or series reservation using an Outlook template using multiple rooms and then edited the reservation, the following issues presented on the calendar view:

  • timelines displayed incorrectly
  • some booked locations did not have edit/delete icons or did not display on the grid
  • deleting a location caused others to disappear from the grid

Fix: Modified the display to ensure that the above issues are resolved and users can see all booked locations on the grid as designed.

24510

Changing Buildings Incorrectly Changed Time Zone for Outlook Bookings

When users changed the building for an Outlook reservation to one in a different time zone, the time zone for the event incorrectly changed to the new building time zone.

Fix: Modified functionality to ensure that time zones on Outlook meetings do not change when buildings are changed.

24629

Delete Icon Missing for Newly Added Locations for Outlook Bookings

When users added a location to an Outlook booking, the Delete icon was missing next to it. 

Fix: Modified the display so that the Delete icon shows next to newly added locations.
