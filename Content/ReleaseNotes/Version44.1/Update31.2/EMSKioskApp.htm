Provisioning

 

Unable to Provision EMS Kiosk App Under Certain Profiles Due to Amount of Room Diagrams

When attempting to choose a profile, the profile modal page in the EMS Kiosk App kept spinning until it timed out if the profile contained a high number of room diagrams (more than 250).

Fix: Modified functionality so that users can select a profile regardless of the number of room diagrams (more than 250).

Internal Reference # 42979
(For EMS Internal Use Only)

Duplicate Platform API Requests on Provisioning and Re-authentication

The EMS Kiosk App was executing each Platform API request twice during provisioning and re-authentication, resulting in long provisioning time and potentially timeout.

Fix: Modified functionality so that the EMS Kiosk App executes Platform API calls correctly.

Internal Reference # 43014
(For EMS Internal Use Only)
