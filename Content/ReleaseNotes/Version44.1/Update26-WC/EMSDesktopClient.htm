The Web Client has been updated to use the EMS Desktop Client LDAP system parameter Use Integrated Authentication Only (accessed from System Parameters > Desktop Client tab > System – Use Integrated Authentication Only) .

Fixes

Area

Description

ActiveReports

ActiveReports Library Not Licensed

When users generated a confirmation or report, a message appeared at the bottom of the confirmation or report stating that the document was created using an evaluation version of ActiveReports and that only a licensed user may legally create reports for use in production.

Fix: Modified functionality so that the ActiveReports license is valid and the invalid license message does not appear.

Internal Reference # 38828
(For EMS Internal Use Only)

Attachments

Unable to Attach Files to Reservations in Navigator

EMS Web Client users were unable to attach files to reservations in the Attachments tab of the Navigator.

Fix: Restored functionality so that users can attach files to reservations in the Navigator.

Internal Reference # 39060
(For EMS Internal Use Only)
