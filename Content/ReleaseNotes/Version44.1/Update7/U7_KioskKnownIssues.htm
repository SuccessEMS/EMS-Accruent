Active Indicators Misaligned on Floormaps

Floorplan active indicators are misaligned.

Customers can fix this problem by running this query in the EMS database:

"UPDATE tblImageRoom SET XCoord = XCoord - 10."

You should only have to run this query once to fix the problem.
