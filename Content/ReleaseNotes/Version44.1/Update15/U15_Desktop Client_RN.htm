Fixes 

Area

Description

Time Zone

 

Time Zone Was Missing for La Paz, Baja California Sur Mexico

The Daylight Savings Time did not exist in the system and not applying correctly to the GMT time zone.

Fix: Modified functionality so that the Daylight Savings Time is applied correctly. 

Internal Reference # 29498

SIS

 

Updated Child Cross-Listed Courses (SIS) Were Not Synchronizing Changes to EMS 

Cross-listed course locations in the SIS are being bounced back to the parent location on auto sync.

Fix: Modified functionality so that cross-listed course locations do not pull from parent when Requires different room is selected.

Internal Reference # 31799

Create a Reservation

 

Category State Default Value Was Not Set

The Category on a newly created reservation using the unauthenticated process template was not reflecting the default State set. 

Fix: Modified functionality so that the state shows the set default.

Internal Reference # 28980
