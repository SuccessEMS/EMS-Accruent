Restrict Access to Unauthenticated View of Reservation Summary

Through the new View Reservation Summary Security Role, Administrators can now limit unauthenticated access to the View Reservation Summary page. See Parameter Details here.

Internal Reference # 37884
(For EMS Internal Use Only)

Restrict Access For ICS Files to Authorized Users

Requests to send an .ics calendar file are validated so that only hosts, delegates, and those linked by external reference can send meeting details through an .ics calendar file. Attempts by any other user will receive a "not authorized" error.

Internal Reference # 38023
(For EMS Internal Use Only)

Search

 

Minimum Character Count for Attendee Search

New parameter to set the minimum character count to initiate type ahead searches for attendees on Calendar-enabled templates and reservations. The default minimum character count to initial the attendee lookup is set to three (3) characters. See Parameter Details here.

Internal Reference # 38462
(For EMS Internal Use Only)

Configurable Inactivity Setting

Administrators can now configure the duration after the last keystroke entry in a search field before triggering a search call for attendees on Calendar-enabled templates and reservations. See Parameter Details here.
