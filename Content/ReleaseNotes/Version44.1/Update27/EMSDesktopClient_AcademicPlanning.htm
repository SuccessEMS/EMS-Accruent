Area

Navigator >
Room Change Wizard

Unable to Use Room Change Wizard on Child Cross-Listed Courses in Navigator

Users were unable to update a child cross-listed course through the Room Change Wizard (Navigator > Tools > Wizard > Change Rooms) if the parent cross-listed course was associated with more than one child course.

Fix: Modified functionality so that users can successfully update child cross-listed courses through the Room Change Wizard when the parent cross-listed course is associated with multiple child courses.

Internal Reference # 39540
(For EMS Internal Use Only)

Campus Planning Interface (CPI)

Academic Browser Did Not Display the Active Scenario in the Campus Planning Interface

The Academic Browser displayed the SIS Scenario instead of the assigned Active Scenario.

Fix: Modified Functionality so that the Academic Browser displays the “Active Scenario” assigned to the term.
