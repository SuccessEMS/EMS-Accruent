Fixes

AREA DESCRIPTION

Secondary Authentication

Secondary Authentication field not clearing

When users entered incorrect values in the EMS Room Sign App, the secondary authentication fields were not clearing.

Fix: Modified functionality so that fields can be cleared when incorrect values are entered into the EMS Room Sign App.

Internal Reference # 36612
(For Internal Use Only)
 

ArmorActive enclosure was not in sync with EMS Room Sign App

The ArmorActive LED light was not in sync with the EMS Room Sign App status after the device polling interval time passed. The EMS Room Sign App was not triggering a sound when the device polling interval time passed.

Fix: Modified functionality so that the EMS Room Sign App triggers a sound when the device polling interval time passes. This results in the enclosure LED light syncing with the EMS Room Sign App status.

Internal Reference # 36960
For Internal Use Only)
 

Rooms list available at login regardless of whether profile was selected

When signing into the EMS Room Sign App, a room list displayed, regardless of whether the user selected a profile.

Fix: Modified functionality so that the room and profile drop-down choices reset after closing the modal popup.

(For Internal Use Only)
