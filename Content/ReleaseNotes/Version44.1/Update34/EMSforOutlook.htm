Resolve Conflict

Could not add room to reservation-plus sign missing

When creating a meeting in EMS for Outlook, users were unable to resolve a room conflict when they selected an available room, and then changed the time in the drop-down list to a time when the room was not available. After changing the time, the Resolve Conflicts window would appear, but the Add (+) buttons were missing, and users were unable to select a different room.

Fix: Modified functionality so that users can resolve room conflicts after modifying the meeting time in the EMS for Outlook time drop-down list.

Internal Reference # EMS-2318
(For Internal Use Only)

 

Create Reservation

Drag-and-Drop did not update time for meetings in Beijing time zone

For meetings in the Beijing time zone, when users modified the meeting time using drag-and-drop in the Outlook calendar, the EMS for Outlook add-in did not update the time in the database if the change was made on the same day.

Fix: Modified functionality so that the database updates when users in the Beijing time zone make changes via drag-and-drop to the meeting time.

Internal Reference # 42190
(For Internal Use Only)
