Fixes

Area

Description

Data Connectors

Data connector timed out

Connectors timed out when there was a large amount of data on them.

Fix: Modified functionality so that the data connectors can handle large data without timing out.

Internal Reference # 27175
(For Internal Use Only)

Data Connectors

Data connectors were not updating successfully

For users with a larger number of connectors configured and running (>20), connectors were not updating successfully over time.

For example: 

  • 04/08/2017 — 22 Connectors Updated

  • 04/09/2017 — 22 Connectors Updated

  • 04/10/2017 — 22 Connectors Updated

  • 04/11/2017 — 20 Connectors Updated

  • 04/12/2017 — 12 Connectors Updated

Fix: Modified functionality so that all connectors update successfully over time.

Internal Reference # 30219
(For Internal Use Only)

iCal

 

iCal feeds were not updating properly

Changes to events in EMS Master Calendar were not correctly reflecting in Microsoft Outlook using iCal feeds.

Fix: Modified functionality so that changes made in EMS Master Calendar are reflected using iCal feeds.

Internal References # 32500 & 33030
(For Internal Use Only)

RSS Feeds

 

Following EMS Master Calendar updates RSS feeds not working

After updating EMS Master Calendar, the RSS feeds returned no data. In addition, creating new RSS Feeds also returned no data.

Fix: Modified functionality so that RSS feeds pull events that adhere to the RSS Feed Configuration.

Internal Reference # 33227
(For Internal Use Only)
