Area

Description

Email

Unable to send emails using Gmail SMTP email settings

Users were unable to send emails from EMS Desktop Client when using Gmail SMTP email settings. When trying to send an email, they would get the error message:

"The SMTP server requires a secure connection or the client was not authenticated. The server response was: 5.5.1 Authentication Required."

Fix: Modified functionality so that users can send emails from EMS Desktop Client when using Gmail SMTP email settings.

Internal Reference # 41280
(For Internal Use Only)

Software Updates

Older EMS database versions were not auditing the latest ClientConnection in tblServerVersion table

When older EMS Desktop Client versions connected to the database, they were not receiving the alert that a newer version was available. The ClientVersion column in tblServerVersion was being updated with the last version of the Desktop Client used instead of the latest version.

Fix: Modified functionality so that when older Desktop Client versions connect to the database, they receive the alert that a newer version is available.

Internal Reference # 27232
(For Internal Use Only)

Reports

 

Special dates excluded from Utilization Report

Special dates were being excluded from booked time or 'available time' on Room Utilization reports.

Fix: Modified functionality so that the Room Utilization report includes special dates, if there is something booked on those dates.

Internal Reference # 41815
(For Internal Use Only)
