Parameter

 

New Parameter for Custom Icon

A new parameter 'Custom Calendaring Icon,' accessible via the EMS Desktop Client, allows users to replace the default EMS for Outlook logo as seen from within Outlook with a custom icon.

Supported file types are .ico, .jpeg, .jpg, or .png and the recommended size is 32 x 32.

See Also: Configure EMS for Outlook.

(For EMS Internal Use Only)

Conflict Resolution

 

Outlook TBD Rooms and Conflict Resolution

When a user is presented with conflict resolution in the Add-in, they have the option to skip the process and come back to it later. The ability to define an 'Outlook TBD' room for each building in the database has been added (these rooms can be created in bulk via the EMS Desktop Client). It is highly recommended that customers go through this process to keep skipped bookings in the correct timezone, and will also allow for localized notifications and reports to be set up around meetings without locations.

See Also: Resolve Booking Conflicts.

Fixes

Area

Description

Conflict Resolution

Conflict Resolution Modal Was Not Appearing

The Conflict Resolution Modal was not appearing if the reservation had two or more rooms and only one room had a conflict.

Fix: Modified functionality so that the conflict resolution modal appears after user sees the "Checking for conflicts" prompt.

Internal Reference # 34898
(For EMS Internal Use Only)

Service Order Notification

When Editing an Outlook Booking with a Conflict, Setup Type Was Locked

Users who edited a booking with a conflict were seeing empty Setup Type and Attendee values, as well as a Setup Type dropdown list that was not clickable.

Fix: Modified functionality so that users can continue in the reservation process to select a room.

Internal Reference # 35835
(For EMS Internal Use Only)

Adding a Room

Error Received When Trying to Add Rooms

When using the EMS for Outlook Add-in, rooms that were previously unavailable (i.e., the room does not appear on the List View as able to add or does not appear on Schedule View), were blocked and unable to be added. The error, "Error Selecting Room," appeared.

Fix: Modified functionality so that users do not receive an error and the booking can be completed with the desired room.

Internal Reference # 35967
(For EMS Internal Use Only)

Authentication

 

Security Template Was Not Being Assigned During First-Time User Authentication

When authenticating a first-time user in EMS for Outlook, users were receiving an "invalid URL" message.

Fix: Modified functionality so that security and process templates are assigned to the user and authentication can be completed.
