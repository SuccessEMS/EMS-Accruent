Fixes

Area

Description

User-Defined Fields (UDFs)

Error Received When Creating Recurring Reservation on Exchange-Enabled Template

Users creating a recurring reservation through the EMS Web App using an Exchange-enabled template were receiving an error. As a result, the reservation was not being sent to Outlook calendars and attendees were removed.

Fix: Modified functionality so that the reservation can be completed.

Internal Reference # 34514
(For EMS Internal Use Only)

Security

XSS Security Vulnerability Identified

Addressed a security vulnerability for cross site scripting on accountmanagement.aspx and default.aspx pages.

Internal Reference # 35905
(For EMS Internal Use Only)

Timeout

Timeout Error in EMS Web App

Users were experiencing a timeout error on the default.aspx page of the EMS Web App.

Fix: Modified functionality so that users do not experience a timeout error.

Internal Reference # 36017
(For EMS Internal Use Only)

Manage Services

 

Services Were Being Added to all Bookings, Regardless of Selection

When adding services to a booking, the services were being added to each booking, regardless of which bookings were being selected.

Fix: Modified functionality so that services are added only to the bookings that were selected on the “Apply to Additional Bookings” window.

Internal Reference # 36111
(For EMS Internal Use Only)
