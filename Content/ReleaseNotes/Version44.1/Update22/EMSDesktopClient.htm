Display Event Type Color Outline

A book option is now available that will "Show Event Type color as outline on bookings." This option is available if "booking colors based on" is set to "status." If "booking colors based on" is set to event type color, the "outline" option will be labeled "show status color as outline on bookings."

Area

Description

User-Defined Fields (UDFs)

Unable to Add Previously Existing UDF to Reservation

User-defined fields (UDFs) could no longer be added to a reservation more than once.

Fix: Modified functionality so that users can add a UDF to a reservation more than once.

Internal Reference # 28076
(For EMS Internal Use Only)

Service Order Notification

Selecting Finish on a Service Order Notification Was Resetting State to All

Users who opened an existing Service Order Notification Rule, that included a Filter for States, (but did not make any changes to the rule) were experiencing the Finish button reverting the States filter to All. If the user selected the Cancel button, the existing States filter selection was retained.

Fix: Modified functionality so that when Service Order Rules are opened and reviewed, 'States' filter will retain the item previously selected.
