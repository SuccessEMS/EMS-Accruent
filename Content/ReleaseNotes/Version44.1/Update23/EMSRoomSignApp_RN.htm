Fixes

Area

Description

Parameters

Max Number of Minutes for New Booking Parameter Was Not Being Honored

Users who created a new meeting in the EMS Room Sign App were able to click Adjust and exceed the Max Number of Minutes for New Booking parameter setting.

Internal Reference # 30959
(For EMS Internal Use Only)

Room Capacity

Booking Allowed to Be Submitted with Room Capacity Violation

When requesting a space through the EMS Room Sign App, users were able to violate the allowed attendance for the room.

Fix: Modified functionality so that users cannot reserve a room if number of attendees is beyond the maximum capacity for that room.

Internal Reference # 35837
(For EMS Internal Use Only)

Authentication Text

Random Characters Displayed in the Authentication Text Field for Non-English Alphabet

The authentication text field was not handling non-English alphabets (e.g., Japanese, Chinese, Russian, etc.) properly. Instead, random characters and symbols were displaying.

Fix: Modified functionality so that the authentication text field displays non-English alphabets correctly.

Internal Reference # 36027
(For EMS Internal Use Only)
