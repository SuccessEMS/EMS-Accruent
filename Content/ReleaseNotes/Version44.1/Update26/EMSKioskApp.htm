Description

Make a Reservation

Unable to Close Floor Filter

Resolved a display issue on Make a Reservation view where if the profile had more than 18 floors available, the buttons to select or cancel the selection did not display.

Fix: Modified functionality so that the Select and Cancel buttons always display and are actionable within the floor filter modal.

Internal Reference # 37325
(For EMS Internal Use Only)

Authentication
Log In

Incorrect Reservations Showing After Logging In with Secondary Authentication

For EMS Kiosks with the Security parameter set to use LDAP for Secondary User Authentication Type for Badge Authentication, when users selected "I don't have my badge" at the login screen and entered their LDAP credentials, users were shown another person's reservations instead of their own.

Fix: Restored functionality so that when users log in with their LDAP credentials for secondary authentication, users are shown the correct reservations.

Internal Reference # 39513
(For EMS Internal Use Only)
