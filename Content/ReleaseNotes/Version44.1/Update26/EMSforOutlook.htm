Description

Attendee Updates

Reduced the Number Automatic Updates Sent to Attendees

Modifications were made to EMS for Outlook to reduce the number of updates that are automatically sent to attendees. When users make changes to services or other non-critical meeting information, such as the event type, automatic updates will not be sent to attendees.

Internal Reference # 36568
(For EMS Internal Use Only)

Fix

Area

Description

Create or Edit Reservation

EMS for Outlook Add-in Displayed Services for Only One Location

When creating a reservation with multiple rooms in different locations (e.g. New York and London), if services were added to more than one room, the EMS for Outlook add-in would only show services for one room (e.g. rooms in both New York and London had catering services added, but only the services for the London room were displayed in the EMS for Outlook add-in).

Fix: Modified functionality so that when services are added to more than one room in a reservation, all the services added to rooms for that reservation display in the EMS for Outlook add-in.

Internal Reference # 38020
(For EMS Internal Use Only)
