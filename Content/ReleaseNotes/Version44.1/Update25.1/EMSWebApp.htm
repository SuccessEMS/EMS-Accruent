Create a Reservation

Internal Server Error Occurred When Creating a Reservation

For customers upgrading to Update 25 from Update 16 or prior, or for customers performing new installs of Update 25, an internal server error occurred when users clicked Create Reservation, and users did not progress to the Reservation Summary page. Although it appeared that the reservation was not created, the reservation was created. However, because users were never taken to the Reservation Summary page, they would create the reservation again.

Fix: Restored functionality so that when users click Create Reservation, they are taken to the Reservation Summary page.
