Enhancements

Area

Description

Provisioning Screen

Search Field Added to Provisioning Screen

A search field has been added to the Provisioning Screen that allows an Administrator to search for the desired profile (rather than scrolling). This is especially important for customers adopting the EMS Kiosk App on a large scale.

Internal Reference # 40858
(For EMS Internal Use Only)

Authentication

Single Authentication Can Access All Authenticated Functions

Users only need to present authentication credentials once within a session to access areas of the EMS Kiosk App that require authentication.

Internal Reference # 37904
(For EMS Internal Use Only)

My Reservations

Past Bookings Drop From My Reservations View

Once a booking end-time has passed, the reservation will drop from the My Reservations view.

Internal Reference # 40106

Fixes

Area

Description

Make a Reservation

Required Rooms Removed From EUPT Room Still Displayed as Available

When the Enable Everyday User Process Template Security parameter is set to YES and a required room is removed from Available rooms, the room still displayed as Available.

Fix: Modified functionality so that a removed EUPT room does not display.

Internal Reference # 32352
(For EMS Internal Use Only)

Building Closed Screen Did Not Display Future Closures

If the "Start Date Allowed for New Reservation" was configured for the same amount of days as the "Number of Future Concurrent Days" and the building was closed on that day, rooms were still displaying as available.

Fix: Modified functionality so that the Building Closed screen displays and No Rooms are available.

Internal Reference # 33003
(For EMS Internal Use Only)

Building Closed Screen Appeared if Closed All Day on the Previous Day (Web Only)

The Building Closed screen was appearing if the building was Closed All Day on the day before. This is a Web-only issue.

Fix: Modified functionality so that users are able to successfully search and book spaces.

Internal Reference # 34384
(For EMS Internal Use Only)

On Profiles with 2,000+ Rooms, All Rooms Were Displaying As Unavailable on Floorplan

Newly mapped locations on floorplan displayed with "unavailable" indicators on profiles with more than 2,000 rooms.

Fix: Modified functionality so that room availability on the floorplan corresponds with reservations for the required date.

Internal Reference # 36835
(For EMS Internal Use Only)
 

List View Displayed After Reserving a Room on the Profile with 2,000+ Rooms Added

When creating a new or editing an existing Kiosk profile with more than 2,000 rooms, the Room List persisted instead of the Confirmation view being displayed.

Fix: Modified functionality so that the Confirmation view displays after reserving a Room and the Room List disappears.

Internal Reference # 36837
(For EMS Internal Use Only)

Session Timeout

For Already Expired Sessions, Session Timeout Modal Appeared

Session Timeout modal appeared for sessions that had already expired.

Fix: Modified functionality so that Session Timeout modal only appears when session is active.

Internal Reference # 41979
(For EMS Internal Use Only)

Today's Events

Scroll Not Working With Multiple Options

Users were unable to scroll through multiple options in the EMS Kiosk App.

Fix: Modified functionality so that users can scroll through multiple options.

Internal Reference # 32567
(For EMS Internal Use Only)

Start / End Times

Unable to Change Overnight Start/End Times in the Picker a Second Time

When trying to edit overnight start/end times for a second time, users were receiving errors that they cannot select times in the past.

Fix: Modified functionality so that users can should change overnight Start/End times in the picker multiple times.

Internal Reference # 34413
(For EMS Internal Use Only)

Locate a Space

Component Rooms Displayed as Available When Combo Was Booked

When combo rooms were booked, they still appeared as available on the Locate Space screen.

Fix: Modified functionality so that if a combo room is booked, the component rooms appear as unavailable.

Internal Reference # 33144
(For EMS Internal Use Only)

Locate a Person

Groups Were Incorrectly Ordered

When locating a person, users were experiencing search results that were incorrectly ordered.

Fix: Modified functionality so that results are ordered by Group Name in ascending alphabetical order.

Internal Reference # 33241
(For EMS Internal Use Only)

Locate a Person Displayed Inactive Users

Locate a Person displayed both active and inactive users from database.

Fix: Modified functionality so that Locate a Person only displays active persons.

Internal Reference # 41050
(For EMS Internal Use Only)

Floor Map Displayed as Max Zoomed Out When Logged in as a Different Group

When creating a reservation while logged in as a different group, the Floor Map displayed as max zoomed out.

Fix: Modified functionality so that the Floor Map displays properly.

Internal Reference # 42247
(For EMS Internal Use Only)

Incorrect Search Text Did Not Trigger an Error Message

From the Locate a Person page, incorrect text inserted into the search did not trigger an error message.

Fix: Modified functionality so that the search functions properly.

Internal Reference # 42253
(For EMS Internal Use Only)

Searching With Invalid Information Returned Group Results

After performing a valid search on the Locate a Person page, performing an invalid search returned group results.

Fix: Modified functionality so that searching by invalid information triggers an error message.

Internal Reference # 42427
(For EMS Internal Use Only)

Translations

Language Reverted to Default

Language was reverting to default as soon as the session timeout screen appeared, regardless of whether the user chose to continue their session.

Fix: Modified functionality so that language does not revert back to default when the session timeout screen appears if a user has chosen to continue their session.

Internal Reference # 34245
(For EMS Internal Use Only)

My Reservations

Component Rooms Displayed as Separate Bookings When Combo Was Booked

When a combo room was booked, the component rooms were displayed as separate bookings that could be canceled.

Fix: Modified functionality so that component room bookings cannot be canceled when combo rooms are booked.

Internal Reference # 34540
(For EMS Internal Use Only)

SVG

SVG Indicators Not Displaying (IE11)

When using Internet Explorer 11 (IE11), SVG indicators were not displaying for pages that contained a map.

Fix: Modified functionality so that SVG indicators display in IE11 on all pages that contain a map.

Internal Reference # 35163
(For EMS Internal Use Only)

Today's Events

Today's Events Not Displaying All Events

When viewing Today's Events, only a subset of events were displaying.

Fix: Modified functionality so that all events display on the Today's Events screen.

Internal Reference # 36575
(For EMS Internal Use Only)

Create a Reservation

Messaging Formatting Issue For Reservations That Span Overnight

When users created a reservation that ended on the next day, a confusing message appeared that only displayed the start date despite the reservation ending on next day.

Fix: Modified functionality so that when users create a booking that spans overnight, both the start date and end date are displayed.

Internal Reference # 36988
(For EMS Internal Use Only)

Floor Plan

Floor Plan Indicators Misaligned on Zoom In/Out

When users zoomed in/out on a floor map in the EMS Kiosk App, indicators could become misaligned.

Fix: Modified functionality so that indicators remain static to location spot on map.

Internal Reference # 37009
(For EMS Internal Use Only)

Default View

Default View Remained on Floor Map View

Despite the parameter "Default search=Floor Map" being disabled, the default view remained on the Floor Map view.

Fix: Modified functionality so that the default view displays the list view once parameter is disabled.

Internal Reference # 37049
(For EMS Internal Use Only)

When users attempted to Choose a Floor from Today's Events, the floors displayed incorrectly.

Internal Reference # 41976
(For EMS Internal Use Only)
