Fix

Area

Description

Modify Booking Details

Expert User Permissions: Modify Booking Details

The Modify Booking Details expert user process was not behaving as expected. Users could not add/edit/delete appropriate categories and resources without additional permissions.

Fix: Modified functionality so that the Modify Booking Details security process allows expert users to add/edit/delete booking details (categories) and booking detail items (resources) on any booking they have permission to access.

Internal Reference # 42117
(For EMS Internal Use Only)
