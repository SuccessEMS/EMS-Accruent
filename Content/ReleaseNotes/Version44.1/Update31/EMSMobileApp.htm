Area

Create a Booking

EMS Mobile App Was Not Following Template Booking Rules Correctly

When the booking rule Only allow new bookings within Range of this Many Days was set to a number less than 24, users could not create a booking past the current date in the EMS Mobile App.

Fix: Modified functionality so that the EMS Mobile App applies booking rules correctly.
