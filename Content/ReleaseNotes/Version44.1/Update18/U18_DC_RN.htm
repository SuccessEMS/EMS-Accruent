Booking History

Booking History Tab Displaying Weird Behavior

When users changed the status of a booking in the Change Status wizard, the reconfirm date was being disabled. The Booking History tab was then displaying the new reconfirm date value as 1/1/1900.

Fix: Modified functionality so that the Booking History tab no longer displays the reconfirm date.

EMS Internal Reference # 29040
(For EMS Internal Use Only.)

Performance

Desktop Web Deploy Client Slow Performance Issue

Clients with certain proxy servers were experiencing performance issues in EMS Desktop Web Deploy Client. Screens were taking too long to open.

Fix: Modified functionality so that users experience a faster response within the EMS Desktop Client.

Internal Reference # 33328
(For EMS Internal Use Only.)
