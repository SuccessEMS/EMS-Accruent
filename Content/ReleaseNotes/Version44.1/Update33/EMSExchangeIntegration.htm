Internal Reference # EMS-2119
(For Internal Use Only)

EMS Web App > Create a Reservation

Exchange availability breaking for environments configured with multiple domains under one tenant

For environments configured with multiple domains under one tenant, Exchange availability was breaking sporadically in EMS Web App. When users tried to add attendees to a reservation, they would get the error "There was problem accessing the attendee service," and they would be unable to add attendees.

Fix: Modified functionality so that in environments configured with multiple domains under one tenant, users can successfully add attendees to a reservation.
