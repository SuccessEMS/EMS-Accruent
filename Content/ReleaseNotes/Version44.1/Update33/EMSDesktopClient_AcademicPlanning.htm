Reporting

Total and average for "All Buildings" were incorrect on Classroom Utilization Report

When running a Classroom Utilization Report for more than one building, the report gave the incorrect number for the Total for all Buildings and the Average for all Buildings. Instead of totaling and averaging all buildings, the report listed the total and average for the last building on the report.

Fix: Modified functionality so that the Classroom Utilization Report lists the correct total and average for the Total for all Buildings and the Average for all Buildings.

Campus Planning Interface (CPI)

Location column was empty for courses with "Publish in SIS Room" selected as preference

In the CPI Academic Browser, the Location column was empty for courses that had Publish in SIS Room selected as a preference in the EMS Desktop Client (Academic Browser > Set Course Preferences > > Preferences tab > Publish in SIS Room option).

Fix: Modified functionality so that the Location column displays the room locations for courses with Publish in SIS Room selected as a preference.
