Internal Reference # EMS-5202
(For Internal Use Only)

Create Reservation, 2-Way Sync

2-Way Sync – Ability to sync reservations created in EMS Room Sign App with Exchange

For unauthenticated users, when they create Exchange integrated reservations in EMS Room Sign App, the reservations will be synced with Exchange room/resource calendars. This enhancement enables the room/resource calendars to accurately reflect room availability.

For authenticated users, when they create Exchange reservations on the Room Sign App, the reservations will be synced with Exchange and will appear in their mailbox calendars and the room/resource calendars to accurately reflect availability.

Internal Reference # EMS-5056 and EMS-5058
(For Internal Use Only)

Edit Reservation, 2‑Way Sync

2-Way Sync – Extend Exchange bookings

For single location Exchange bookings that are created in EMS Room Sign App, when users extend the end time of a booking, the end time will also be updated in Exchange.

For multiple location Exchange bookings, the extend end time functionality is not available. On the EMS Room Sign App, the Extend button will be grayed out, and if a user taps on the Extend button, the message "Unable to Extend this booking" will be displayed.

Internal Reference # EMS-5068 and EMS-5071
(For Internal Use Only)

End Reservation, 2‑Way Sync

2-Way Sync – End Exchange bookings

Single location Exchange bookings—from the EMS Room Sign App, when a user ends a booking for a single-location reservation, the following will occur:

  • The booking's end time will be updated to the time the user selected the End Now button on the EMS Room Sign App. The booking's end time will be updated in the EMS Room Sign App, the EMS database, the EMS Desktop Client, and the Exchange calendars for the attendees.
  • The booking is removed from the room and resource calendars.
  • The organizer's Exchange calendar will be updated, if the organizer is an everyday user.

Multiple location Exchange bookings—from the EMS Room Sign App, when a user ends a booking that is a part of a multi-location reservation, the following will occur:

  • The booking's end time will be updated in EMS for the room.
  • The room will be removed from the Exchange reservation/appointment.
  • If the room of the booking that was ended was the organizer of the meeting, the next room mailbox in the Exchange reservation will be the organizer.
  • Video Conference host will remain as is.

Internal Reference # EMS-5062 and EMS-5065
(For Internal Use Only)

Checking In, 2‑Way Sync

2-Way Sync – Auto cancel Exchange booking after user fails to check in

Auto cancel functionality is available for Exchange bookings. If auto cancel functionality is enabled and users fail to check into an Exchange booking within the time specified, the booking will be automatically canceled.
