V44.1, Update 14

This section describes updates to your EMS software.

Name: V44.1 Update 14

Release Date: July 31, 2017

Patch: 44.1.14000.201707251

Build #:  44.1.14000.349

Fixes

Area

Description

Optimization

Conflicts for Optimization Scenarios Showed Conflicts That Were Not Actual Conflicts

Conflicts for Optimization scenarios showed courses from all terms that were not actual conflicts. 

Fix: Modified functionality so that conflicts shown are actual conflicts. 

Internal Reference # 21142

Reporting

Utilization Report Error

When users tried to access the Room Utilization report, they received an error message stating, "Conversion from string '0.9' to type 'Double' is not valid. Input string was not in a correct format."  

Fix: Modified functionality so that the Room Utilization report can be run without errors. 

Internal Reference # 29243

EMS Web Deploy

EMS Web Deploy Was Not Automatically Updating

When users launched the EMS Web-deploy Desktop Client, automatic updates were not taking place. If they attempted to run the AutoUpdate.exe manually, they received the following error, "EMS is having trouble connecting," and the EMS Desktop Client stopped working entirely. Users on version 12 or later needs to upgrade to Update 14. 

Fix: Modified functionality so that the EMS Web-deploy automatically updates when clicked. 

Internal Reference # 31453
