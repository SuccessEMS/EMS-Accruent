Edit Video Conference Reservation > Add Room 

Unable to Add Rooms to Video Conferences

When editing an existing video conference reservation to add a room, the Host Location drop-down field only presented the current room and a "(none) option, so users were unable to set a different room as the host. Selecting the host room produced multiple hosts.  

Fix: Modified field functionality so that additional rooms display in the drop-down and users can select a different host room without duplication.
