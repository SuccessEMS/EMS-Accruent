Area

Description

Event Sheet Report

Missing Time Stamp on Event Sheet

Date was displaying without the time. Date, time, and initials of user have been added to the date/time stamp.

Fix: Modified functionality so that date, time, and initials of user are visible on the date/time stamp.

Internal Reference # 37855
(For EMS Internal Use Only)

Grouping Not Displayed on Event Sheet

Grouping was not displayed on the Event Sheet.

Fix: Modified functionality so that the grouping is displayed on the Event Sheet.

Internal Reference # 37856
(For EMS Internal Use Only)

Pulled Service Orders Appeared Across Multiple Sheets

Users pulling multiple service orders were experiencing the service orders displaying on multiple sheets.

Fix: Modified functionality so that when pulling multiple service orders, there is a separate sheet for each service order.

Internal Reference # 37858
(For EMS Internal Use Only)

Event Location

Header Message Not Displaying Properly

The Event Location box was preventing the Header Message from displaying properly.

Fix: Modified functionality so that the full header message displays.

Internal Reference # 37857
(For EMS Internal Use Only)
 

Filters Did Not Function Properly for Single Day Searches

Users were unable to filter by Event Name or Single/Multiple Reservation ID when performing single-day searches.

Fix: Modified functionality so that users can use event name, single or multiple reservation ID filters for single date searches.

Internal Reference # 37859
(For EMS Internal Use Only)
 

Users Could Not Search for a Contact Without Providing a Group

Users were unable to search for a contact independent from a group. They were forced to enter/search for a department prior to searching for a contact.

Fix: Modified functionality so that users can search for a contact without identifying a group first.

Internal Reference # 37860
(For EMS Internal Use Only)
