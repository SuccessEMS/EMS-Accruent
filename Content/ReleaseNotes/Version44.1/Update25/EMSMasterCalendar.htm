Area

Description

Connector Service

Master Calendar TBLEventDetailHistory is logging ALL events

All events that the Master Calendar connector service was sending to EMS Master Calendar were being added to the tblEventDetailHistory table.

Fix: Modified functionality so that only modified events are added to the tblEventDetailHistory table.

Internal Reference # 37087
(For EMS Internal Use Only)

Image and Image Alt Text

 

Images and Image Alt Tags Were Not Displaying for All Admins from the Event Information Page

For events created in EMS and brought over to Master Calendar through the connector, when an admin added an image and an image alt tag to an event, the EventInfoRequest.aspx page did not display the image and image alt tag to any admins except the admin who originally added the image and alt tag.

Fix: Modified functionality so that the EventInfoRequest.aspx page displays the image and image alt tag to all admins.

Internal Reference # 38350
(For EMS Internal Use Only)
