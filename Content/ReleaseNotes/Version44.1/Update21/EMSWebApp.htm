Fixes

Area

Description

Special Characters

 

Group Search in Browse People Did Not Return Results with Apostrophe in Name

The group search filter was not handling special characters.

Fix: Modified functionality so that the group search returns results with special characters in the name.

Internal Reference # 34128
(For EMS Internal Use Only)

ICS File

 

Incorrect Date, Time, and Time Zone Displayed in ICS File

Users who created a reservation from EMS Desktop Client or the EMS Web App and clicked Add to My Calendar received an ICS file with the wrong date and time.

Fix: Modified functionality so that the ICS file is correct.

Internal Reference # 34378
(For EMS Internal Use Only)

UDFs

 

 

 

User-Defined Fields Not Working As Expected

After inserting text into the user-defined field, the red box did not show the requirement was met. This happened for all other required fields.

Fix: Modified functionality so that the red box turns grey once text has been inserted, indicating the requirement has been met.

Internal Reference # 30779
(For EMS Internal Use Only)

Adding a Visitor from the EMS Web App Required an Extra Click

Visitor page was not remaining open when making a reservation in the EMS Web App, forcing users to add Visitors one at a time.

Fix: Modified functionality so that the Visitor page remains open, allowing users to add multiple Visitors to the reservation.

Internal Reference # 35410
(For EMS Internal Use Only)

Billing Reference Allocation Numbers Were Not Parsing Ampersands (&) Correctly

Billing References Numbers that included an ampersand and had been set to a Group were not displaying correctly in the EMS Web App.

Fix: Modified functionality so that special characters display correctly for Cost Centers/PO/Billing References.

Internal Reference # 35411
(For EMS Internal Use Only)

Billing Reference Was Not Updating When Group Was Updated

Everyday users who updated a group to a new group, were experiencing default billing reference numbers that were not populating as expected. The old billing references remained.

Fix: Modified functionality so that billing references change to reflect the Group's billing reference.

Internal Reference # 35412
(For EMS Internal Use Only)

New Booking Cutoff Time

 

Server Time Zone Was Being Honored by "Use New Booking Cutoff Time"

When creating a reservation in the EMS Web App, regardless of which building a user chose, cutoff times were being created from the server's Time Zone.

Fix: Modified functionality so that cutoff times are created by the building's Time Zone.

Internal Reference # 35408
(For EMS Internal Use Only)

Make a Reservation

 

Users Unable to Book More than One Booking on Template That Allows Multiple Bookings

Users were unable to book more than one booking on templates that were configured for multiple bookings.

Fix: Modified functionality so that users can book multiple bookings on multiple booking templates.

Internal Reference # 34877
(For EMS Internal Use Only)

Add Services

 

Catering/Services Required User to Enter an Estimated Count

When adding a catering resources to a reservation, the estimated count should be populated by the attendance count, if available. If not available, estimated count should have been populated with the default.

Fix: Modified functionality so that the estimated count is populated by the attendance count.

Internal Reference # 35409
(For EMS Internal Use Only)

Attendees

 

Unable to Find Attendees (Mobile Web App)

Users were unable to search for attendees from the room request screen when using a G Suite-enabled template in the Mobile Web App.

Fix: Modified functionality so that G Suite attendees can be found, availability is displayed, and attendees can be invited to the reservation.

Internal Reference # 35049
(For EMS Internal Use Only)

Authentication

 

When SAML Authentication Was Enabled, Native Auth Form Still Appeared

When SAML authentication was enabled, users were still able to login using native authentication credentials.

Fix: Modified functionality so that the native authentication form is hidden.

Internal Reference # 35533
(For EMS Internal Use Only)

Event Name

 

Default Event Name on Templates Was Not Parsing Special Characters Correctly

Users entering an event name with a special character in a process template did not see those characters parsing correctly in the EMS Web App.

Fix: Modified functionality so that special characters render properly.

Internal Reference # 35738
(For EMS Internal Use Only)

Attachments

 

Attachments Cannot Be Downloaded in the EMS Web App

Attachments on the Reservation level were corrupted and not viewable in the EMS Web App.

Fix: Modified functionality so that attachments can open properly.

Internal Reference # 35643
(For EMS Internal Use Only)
