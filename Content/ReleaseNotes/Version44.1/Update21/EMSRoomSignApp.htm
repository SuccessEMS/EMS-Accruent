Enhancements

Area

Description

Time Bar

Time Bar Color Change

The time bar in the EMS Room Sign App will display as green if the room is available and red if it's reserved.

Internal Reference # 35155
(For EMS Internal Use Only)

Aura Firmware Integration

LED Functionality

The new Aura firmware integration provides new LED functionality for selecting a profile, booking/checking into a room, ending or extending a reservation, authentication, as well as landscape/portrait modes.

Internal Reference # 35659
(For EMS Internal Use Only)

Fix

Area

Description

LEDs

LED/Profile Change Issue

The LED states on MIMO devices were getting lost when switching between profiles/rooms after the initial profile had started.

Fix: Modified functionality so that all LED functionality works as expected when using MIMO Hardware.

Internal Reference # 35236
(For EMS Internal Use Only)
