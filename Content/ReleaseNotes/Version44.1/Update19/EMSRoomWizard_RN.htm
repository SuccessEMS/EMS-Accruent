Checking In

Checking in From Any EMS Product Will Register on the EMS Room Wizard Panel

A check-in on any EMS product is registered on the EMS Room Wizard Panel. Once a user checks into a booking using any EMS product, the "Start" button is removed from the Room Wizard panel so that user does not need to check in again from the Room Wizard panel.

Note : The "Start Early" button will still appear before the meeting start time even if the user has checked in from any EMS Product. Room Wizard currently does not provide an API to suppress this button.
