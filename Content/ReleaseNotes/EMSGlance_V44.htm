Glance displayed the booking group and event name by default. Users could designate a booking as private by configuring a status in the EMS Client. Only this single status could be used to designate a private booking.

Enhancement. ​It is now possible to designate a private booking with 'Event Type' as well as status. This enhancement provides more flexibility in configuring EMS Glance.

Requires latest Ems.Glance.dll and EMS Glance web installer
