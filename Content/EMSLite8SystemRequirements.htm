EMS Lite 8 System Requirements

EMS Desktop Client

 

Operating System

Windows 8 (32-bit or 64-bit)
Windows 7 (32-bit or 64-bit)
Windows Vista
Windows XP Pro x/SP3

Minimum System Requirements

Processor: 750 MHz Pentium III-compatible (or faster)

Memory: 1 GB for Windows 8/7/Vista
512 MB for Windows XP or more

Hard-Disk Space: 100 MB
Software: .NET Framework 2.0
Video: SVGA 1024 x 768 or higher resolution
Network Card: 100 / 1000 Mbps Ethernet, switched

 

Database Server

Supported Platforms

Windows 2012 Server
Windows 2008 Server R2
Windows 2003 Server R2
Windows Small Business Server (SBS) 2008/2011

Minimum System Requirements

Processor: 1 GHz or faster

Memory: 1 GB or more

Hard-Disk Space: 600 MB
Software: Microsoft SQL Server 2008 R2
Network Card: 100 / 1000 Mbps Ethernet, switched

These are the minimum system requirements for the optional web component, Virtual EMS (VEMS).

Web Applications (Virtual EMS/EMS Web Client) – Client

Operating System

Any system that supports the required Internet browser software.

Minimum System Requirements

Processor: 450 MHz Pentium III- compatible or faster (or comparable processor)
Memory: 256 MB or more
Software: Internet Explorer 10.0/9.0/8.0/7.0, Firefox, Google Chrome, or Safari
Video: SVGA 1024 x 768 or higher resolution
Network Card: 10/100/1000 Mbps Ethernet (or other Internet connectivity)
