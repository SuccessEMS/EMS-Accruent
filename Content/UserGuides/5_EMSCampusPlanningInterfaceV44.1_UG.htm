EMS Campus Planning Interface (V44.1) Everyday User Guide

EMS Campus Planning Interface (CPI) is a Web-based interface that academic clients typically use to review course schedule data and communicate changes to the Domain Scheduler. This optional interface is designed for academic users to enable them to manage space on-campus.
